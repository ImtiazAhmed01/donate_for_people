Important FAQ

What is the fullform of 'DOM'?

Document Object Module

What are the differences between querySelector and querySelectorAll?

querySelector returns the first element that matches a specified CSS selector, querySelectorAll returns a static NodeList of all elements that match the specified CSS selector.

What is event delegation and how does it work?

Event delegation involves adding a single event listener to a parent element to manage events for its children. It takes advantage of event bubbling.
